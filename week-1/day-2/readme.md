1. What is Salesforce Platform?
The Salesforce Platform is a cloud-based platform used to build, manage,and customize business applications like CRM, sales,
 service, and automation without needing heavy hardware or software installation.

2. Explain
App:
An App in Salesforce is a collection of tabs, objects, and tools designed for a specific business process.
Object:
An Object in Salesforce is a database table used to store information such as customer details, employee data, or sales
records.
Tab:
A Tab in Salesforce is a user interface element that helps users access objects, records, and applications easily.

3. Difference: Configuration vs Coding
Configuration	Coding:
Uses clicks and settings	Uses programming languages
No technical coding knowledge required	Requires developer knowledge
Faster and easier customization	Used for complex customization
Example: Creating fields and workflows	Example: Apex and Lightning Components
